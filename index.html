BlazePHP

BlazePHP is a blazing-fast, multilingual, ultra-secure PHP framework designed for modern APIs and full-stack development.

Key Features

  • RESTful API with JWT authentication
  • RBAC (role-based access control)
  • Plugin system and Live Code Playground
  • AI Developer Assistant
  • Multilingual support with auto-language resolution
  • Vue Admin panel
  • Secure Middleware: CSRF, XSS, RateLimiter, Firewall
  • Docker + CI/CD + PHPUnit

Quick Setup

composer install
cp config/env.example.php config/env.php
sh install/setup.sh
mysql -u root -p blazephp < install/setup.sql
php -S localhost:8000 -t public

Vue Admin

cd frontend/vue-admin
npm install
npm run serve

Docker

docker-compose up -d --build

Testing

vendor/bin/phpunit
